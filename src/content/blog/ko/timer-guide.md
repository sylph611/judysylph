## 왜 타이머가 필요한가?

### 파킨슨의 법칙
> "일은 주어진 시간을 다 채운다"

시간 제한을 두면 같은 작업도 더 빨리 완료할 수 있습니다.

### 타이머의 효과
- 집중력 향상
- 과로 방지
- 작업 속도 측정
- 시간 감각 개선

## 뽀모도로 기법

### 기본 규칙
1. **25분** 집중 작업
2. **5분** 짧은 휴식
3. 4회 반복 후 **15-30분** 긴 휴식

### 장점
- 번아웃 방지
- 꾸준한 집중력 유지
- 완료 성취감
- 작업량 측정 가능

### 변형
| 유형 | 집중 | 휴식 |
|------|------|------|
| 클래식 | 25분 | 5분 |
| 52/17 | 52분 | 17분 |
| 90/20 | 90분 | 20분 |
| 마이크로 | 15분 | 3분 |

## 활용 분야

### 공부
- 시험 준비
- 독서
- 온라인 강의
- 암기 학습

### 업무
- 보고서 작성
- 이메일 처리
- 회의 시간 관리
- 프로젝트 작업

### 운동
- HIIT (고강도 인터벌)
- 플랭크
- 스트레칭
- 러닝 인터벌

### 일상
- 청소
- 요리
- 휴식
- 게임 시간 제한

## 효과적인 타이머 사용법

### 1. 작업 정의
시작 전 무엇을 할지 명확히 정하기
```
✓ "보고서 서론 작성" (구체적)
✗ "일하기" (모호함)
```

### 2. 방해 요소 제거
- 스마트폰 무음/뒤집기
- 알림 끄기
- 문 닫기
- 집중 음악 (선택)

### 3. 휴식 시간 지키기
- 실제로 휴식하기
- 스트레칭
- 물 마시기
- 눈 휴식

### 4. 기록하기
완료한 뽀모도로 수 기록
- 일일 목표 설정
- 생산성 패턴 파악

## 타이머 유형

### 카운트다운
- 정해진 시간 후 알림
- 뽀모도로, 요리, 운동

### 스톱워치
- 경과 시간 측정
- 작업 시간 기록

### 인터벌 타이머
- 작업/휴식 자동 반복
- HIIT, 회의

## 추천 시간대

### 집중 작업
| 작업 유형 | 권장 시간 |
|-----------|-----------|
| 창의적 작업 | 45-90분 |
| 단순 반복 | 25분 |
| 학습 | 50분 |
| 회의 | 30-45분 |

### 휴식
| 집중 시간 | 휴식 시간 |
|-----------|-----------|
| 25분 | 5분 |
| 50분 | 10분 |
| 90분 | 15-20분 |
