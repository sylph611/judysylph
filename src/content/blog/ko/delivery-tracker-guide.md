## 택배 배송 조회 방법

택배 배송 현황을 실시간으로 확인하는 방법을 알아봅니다.

## 배송 상태 용어

### 일반적인 배송 단계
| 상태 | 의미 |
|------|------|
| 집하 | 택배사가 물건을 수거함 |
| 간선상차 | 허브 터미널로 이동 중 |
| 간선하차 | 허브 터미널 도착 |
| 배달출발 | 배송 기사 출발 |
| 배달완료 | 수령 완료 |

### 특수 상태
- **배송지연**: 물량 과다, 날씨 등으로 지연
- **주소불명**: 주소 확인 필요
- **부재**: 수령인 부재로 재배송 예정
- **반송**: 배송 불가로 되돌아감

- 배송 기사 위치 확인 가능

## 배송 조회 팁

### 1. 운송장 번호 확인
```
✓ 숫자만 입력 (하이픈 제외)
✓ 대소문자 주의 (해외 배송)
✓ 복사-붙여넣기 권장
```

### 2. 조회 불가 시
- 발송 후 1-2시간 대기
- 택배사 고객센터 문의
- 판매자에게 확인

### 3. 배송 지연 시
- 추석/설 기간: 3-5일 추가
- 악천후: 1-2일 추가
- 도서산간: 1-3일 추가

## 알림 서비스

### 카카오톡 알림
대부분의 쇼핑몰에서 카카오 알림톡 발송

### 네이버 스마트배송
네이버쇼핑 주문 시 자동 통합 조회

### 쿠팡앱
푸시 알림으로 실시간 배송 현황

## 수령 방법

### 직접 수령
- 가장 확실한 방법
- 부재 시 재배송

### 경비실/무인택배함
- 부재 시 대체 수령
- 분실 위험 주의

### 문 앞 배송
- 사전 요청 필요
- 분실/도난 책임 본인

### 픽업 서비스
- GS25, CU 등 편의점
- 택배 보관함 (네이버플러스)

## 배송 문제 해결

### 배송 지연
1. 택배사 고객센터 문의
2. 배송 기사 직접 연락
3. 판매자에게 문의

### 물품 파손
1. 수령 시 상태 확인
2. 사진/영상 촬영
3. 택배사 클레임 접수
4. 판매자에게 환불/교환 요청

### 오배송
1. 택배사 신고
2. 올바른 주소로 재배송 요청
3. 받은 물품은 반송

### 분실
1. 택배사 분실 신고
2. 경찰 신고 (필요 시)
3. 보상 청구
