## 왜 글자수를 세야 할까요?

글자수와 단어수 카운트는 작가, 마케터, 개발자에게 필수입니다. 플랫폼 제한, SEO 요구사항, 콘텐츠 가이드라인에서 정확한 글자수를 요구합니다.

## 무엇을 카운트하나요?

### 글자(문자)
- 한글, 영문 문자
- 숫자
- 문장부호
- 공백 (대부분 포함)
- 특수문자
- 이모지 (2자 이상으로 계산될 수 있음)

### 단어
공백이나 문장부호로 구분된 문자열의 집합

### 문장
마침표(. ! ?)로 끝나는 단위

### 문단
줄바꿈이나 빈 줄로 구분된 단위

## 플랫폼별 글자수 제한

### SNS
| 플랫폼 | 제한 |
|--------|------|
| 트위터/X | 280자 |
| 페이스북 | 63,206자 |
| 인스타그램 캡션 | 2,200자 |
| 링크드인 | 3,000자 |
| 틱톡 캡션 | 2,200자 |
| 카카오톡 상태 | 60자 |
| 네이버 블로그 제목 | 100자 |

### 메시지
| 플랫폼 | 제한 |
|--------|------|
| SMS (한글) | 70자 |
| SMS (영문) | 160자 |
| 카카오톡 | 10,000자 |
| 이메일 제목 | 50자 권장 |

### SEO
| 요소 | 권장 글자수 |
|------|-------------|
| 타이틀 태그 | 30-40자 (한글) |
| 메타 설명 | 80-120자 (한글) |
| URL 슬러그 | 30자 이하 |
| H1 제목 | 35자 이하 |

## 글자수 카운트 팁

### SNS 글쓰기
- 링크 공간 확보 (단축 URL ~23자)
- 해시태그 글자수 고려
- 멘션(@) 공간 확보
- 이모지는 2자 이상 차지할 수 있음

### SEO 최적화
- 중요 키워드는 앞쪽에 배치
- 픽셀 너비 제한 고려
- 메타 설명에 행동 유도 문구 포함

### 학술/공식 문서
| 유형 | 일반적인 길이 |
|------|---------------|
| 초록 | 300-500자 |
| 에세이 | 2,000-10,000자 |
| 리포트 | 5,000-20,000자 |
| 논문 | 50,000자 이상 |

## 읽기 시간 계산

평균 읽기 속도:
- 성인: 분당 200-300자 (한글)
- 기술 문서: 분당 150-200자
- 훑어읽기: 분당 400-600자

### 간단한 공식
읽기 시간 = 글자수 ÷ 250

1,000자 ≈ 4분 소요
