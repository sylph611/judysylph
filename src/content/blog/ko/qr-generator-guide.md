## QR코드란?

QR코드(Quick Response Code)는 1994년 일본 덴소웨이브가 개발한 2차원 매트릭스 바코드입니다. 스마트폰 카메라로 스캔하면 즉시 정보를 읽을 수 있습니다.

## QR코드의 구조

### 기본 요소
- **위치 탐지 패턴**: 3개의 큰 사각형 (모서리)
- **정렬 패턴**: 작은 사각형 (큰 QR코드에만)
- **타이밍 패턴**: 수평/수직 점선
- **데이터 영역**: 실제 정보 저장

### 오류 정정 레벨
| 레벨 | 복원 능력 | 용도 |
|------|-----------|------|
| L (Low) | 7% | 깨끗한 환경 |
| M (Medium) | 15% | 일반 용도 |
| Q (Quartile) | 25% | 인쇄물 |
| H (High) | 30% | 로고 삽입 시 |

## QR코드 유형

### 1. URL
가장 흔한 사용법
```
https://example.com
```

### 2. 연락처 (vCard)
```
BEGIN:VCARD
FN:홍길동
TEL:010-1234-5678
EMAIL:hong@email.com
END:VCARD
```

### 3. Wi-Fi
자동 연결
```
WIFI:T:WPA;S:네트워크이름;P:비밀번호;;
```

### 4. 이메일
```
mailto:email@example.com?subject=제목&body=내용
```

### 5. 전화
```
tel:010-1234-5678
```

### 6. SMS
```
sms:010-1234-5678?body=메시지
```

### 7. 위치 (지도)
```
geo:37.5665,126.9780
```

## QR코드 디자인 팁

### 1. 대비 유지
- 배경과 전경 색 대비 필수
- 밝은 배경 + 어두운 패턴 권장
- 반전 색상 사용 시 스캔 어려움

### 2. 적절한 크기
| 용도 | 권장 크기 |
|------|-----------|
| 명함 | 2cm × 2cm |
| 포스터 | 5cm × 5cm |
| 현수막 | 10cm+ × 10cm+ |

### 3. 여백 (Quiet Zone)
- QR코드 주변 흰색 여백 필수
- 최소 4셀 너비

### 4. 로고 삽입
- 중앙에 작은 로고 가능
- 오류 정정 H 레벨 사용
- 데이터 영역 30% 미만

## QR코드 스캔 방법

### iOS
- 카메라 앱 열기
- QR코드에 대기
- 알림 탭하여 열기

### Android
- 카메라 앱 (일부 기종)
- Google 렌즈
- 전용 QR 스캔 앱

### 삼성
- 카메라 앱 → 빅스비 비전
- 또는 빠른 설정 패널
